# PoE & PoE 2 Unofficial Launcher Roadmap

이 문서는 앞으로의 개발 우선순위를 관리하기 위한 로드맵입니다.
버전 단위 일정 대신 우선순위로 정렬하며, 완료된 항목은 하단의 `완료됨` 섹션으로 이동합니다.

<!-- issue-sync:omit-start -->

원본: [GitHub Issue #7 - Roadmap](https://github.com/NERDHEAD-lab/POE2-unofficial-launcher/issues/7)

상세 기획은 `docs/roadmap/` 하위 문서로 분리합니다.
체크리스트 오른쪽의 `ref` 링크는 로컬 문서 관리용이며, GitHub Issue #7로 동기화할 때는 제거됩니다.

## P0 - 안정성 및 운영 기반

- [ ] 카카오 게임 실행 자동화 실패 판정 강화 ([ref](roadmap/kakao-automation-failure-detection.md))
  - [ ] 자동화 세션 checkpoint 도입 (`session-started`, `handler-matched`, `user-input-required`, `launch-command-triggered`, `process-start-detected` 등)
  - [ ] 로그인, QR 로그인, 본인인증, 보안센터 등 사용자 입력이 필요한 화면은 timeout 실패 판정에서 제외
  - [ ] 게임 실행 핸드오프 전 실패와 실행 후 조기 종료를 분리
  - [ ] 자동화 실패 확정 시 진단 덤프, 오류 알림, 게임 상태 error 전환을 공통 finalizer로 처리
- [ ] Test Coverage 고도화
- [ ] SonarCloud 적용

## P1 - 실행, 알림, 업데이트

- [ ] 게임 실행 관련
  - [ ] 실행 실패 시 재시도 횟수 설정
- [ ] 알림함
  - [ ] 정보, 경고, 오류 별로 런처 내 알림함 구현
  - [ ] 인게임 채팅 알림 연동
  - [ ] 거래소 판매 알림 연동
- [ ] 업데이트 관련
  - [ ] 업데이트 수동 확인 버튼 추가
  - [ ] 업데이트 팝업 알림 표시하지 않기 옵션 추가
  - [ ] 업데이트 직후 패치노트를 표시하지 않는 옵션 추가
- [ ] 디스코드 연동
  - [ ] 공지사항 연동
  - [ ] 릴리즈 노트 연동
  - [ ] 이슈 트래킹 연동
  - [ ] 오류 보고서 연동
  - [ ] 후원자 신청 기능

## P2 - 사용자 기능 확장

- [ ] 빠른 실행 도구
  - [ ] 게임 및 서비스 별 유용한 링크 추가/관리 기능 제공
  - [ ] 아이콘, 링크, 지원 게임/서비스 선택 기능 제공
  - [ ] 해당 도구를 등록해 둔 런처 사용자 수 통계 제공
- [ ] 아이템 필터 관리
  - [ ] 추가, 제거, 추천 등 관리 기능 제공
  - [ ] 공개 필터에 사용자 커스텀 설정을 자동 override 하는 기능 제공
- [ ] 언어 설정 (i18n)
  - [ ] 영어, 중국어, 일본어 우선 지원
  - [ ] 최초 설치 시 OS 언어 기반 자동 설정 기능 추가
  - [ ] 온보딩 화면에서 수동 언어 설정 기능 추가
- [ ] 후원자 관련
  - [ ] 디스코드를 통해 신청된 후원자 명단을 런처에 표시
- [ ] MAU 측정
  - [ ] Google Analytics Tools 연동 검토
- [ ] 폰트 설치 HKLM → HKCU 전환 — UAC 승인 없이 폰트 설치 ([ref](roadmap/hkcu-font-install.md))

## P3 - 장기 검토

- [ ] 디스코드 Q&A 채널 운영

- [ ] Ultrawide 해상도 패치
  - [ ] own risk 경고 문구 포함
  - [ ] 해당 해상도 사용 시에만 추천
- [ ] 오버레이 관련
  - [ ] 거래소 오버레이
  - [ ] 액트 구간 네비게이션 오버레이
  - [ ] 기타 유용한 페이지를 인게임에서 오버레이로 표시
- [ ] 업데이트 관련
  - [ ] 코드 사인 발급 이후 자동 업데이트 제공
  - [ ] 설치된 게임 자동 업데이트
  - [ ] 게임 버전 정보 callback/broadcast 구조 검토
- [ ] 게임 로그 관리 기능
  - [ ] 게임 로그를 일 단위로 분리
  - [ ] 보관 최대 일수 설정
- [ ] 배경이미지 및 테마 커스텀 기능
- [ ] 광고 표시 기능
  - [ ] 사용자 설정으로 표시 개수와 빈도 선택
