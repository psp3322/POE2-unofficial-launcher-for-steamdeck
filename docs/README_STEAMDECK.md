# POE2 Unofficial Launcher — 스팀덱 베타

[NERDHEAD-lab/POE2-unofficial-launcher](https://github.com/NERDHEAD-lab/POE2-unofficial-launcher)를
스팀덱(Proton)에서 돌릴 수 있게 수정한 포크입니다. 원본과 동일하게 AGPL-3.0 라이선스를 따르며,
원본 프로젝트를 만들어주신 NERDHEAD-lab님께 감사드립니다.

Windows용 런처를 Proton(Wine) 위에서 그대로 실행하는 방식이라, 카카오 로그인 자동화·지정 PC
확인·패치 오류 자동 복구 같은 원본 기능을 스팀덱에서도 그대로 쓸 수 있게 하는 것이 목표입니다.

> ⚠️ **베타입니다.** 실기기 검증이 끝나지 않은 기능이 있습니다. 아래 "알려진 제한"을 먼저 읽어주세요.

## 설치 방법

1. 스팀덱을 **데스크탑 모드**로 전환하세요.
2. 웹브라우저로 이 저장소의 **Releases** 페이지에서 최신 베타 설치파일(`...-sd.X Setup.exe`)을 받으세요.
3. 데스크탑 모드의 Steam에서 **"비 Steam 게임 추가"**로 다운받은 설치파일을 추가하세요.
4. 추가한 게임의 속성 → 호환에서 **Proton 9.0 이상 (또는 GE-Proton 9.x 이상)** 을 강제 지정하세요.
5. 추가한 게임을 실행하면 설치가 자동으로 진행되고, 설치가 끝나면 런처가 자동 실행됩니다.
6. **여기서 끝입니다.** 이후에는 같은 바로가기를 그대로 실행하면 됩니다.
   설치파일이 이미 설치된 런처를 감지하면 재설치 대신 런처를 바로 실행하므로,
   compatdata (랜덤숫자) 폴더를 찾아 바로가기 경로를 수정할 필요가 **없습니다**.
7. 게임 모드로 돌아가서 실행하고, 런처에서 카카오 로그인 후 게임을 시작하세요.
   게임 클라이언트는 같은 Proton 프리픽스 안에 설치됩니다.

## 강제 재설치

런처가 꼬였을 때: 데스크탑 모드에서
`~/.local/share/Steam/steamapps/compatdata/(해당숫자)/pfx/drive_c/users/steamuser/AppData/Local/Programs/`
아래의 런처 폴더를 삭제하고 바로가기를 다시 실행하면 재설치됩니다.

## 스팀덱(Wine) 모드에서 달라지는 것

Wine에는 PowerShell과 UAC가 없어서, 런처가 Wine 환경을 자동 감지해 아래처럼 동작을 바꿉니다.

| 기능 | 스팀덱에서의 동작 |
|---|---|
| UAC 팝업 제거 (RUNASINVOKER) | UAC 자체가 없으므로 항상 "적용됨"으로 처리 (설정 불필요) |
| 게임 프로세스 감시 | PowerShell 대신 `/proc` 스캔으로 대체 (게임 실행/종료 감지 동작) |
| 레지스트리 읽기/쓰기 | `reg.exe` 폴백으로 동작 |
| 예약 패치의 프로세스 강제 종료 | 이름 기반(taskkill /IM)만 동작 |

## 알려진 제한 (베타)

- **시스템 폰트 설치/제거 미지원**: Windows 폰트 API에 의존하는 기능이라 스팀덱에서는 실패합니다. (게임 내 폰트 패치 기능 중 시스템 설치 방식만 해당)
- **공격적(aggressive) 패치 복구 모드의 즉시 강제종료 미지원**: 프로세스 종료를 기다렸다가 복구하는 일반 모드로 동작합니다.
- **창 제목 기반 패치 완료 감지 미동작**: 로그 기반 감지에 의존합니다.
- **지정 PC 자동 처리, 카카오 로그인 자동화**: 코드상 플랫폼 독립적이지만 실기기 검증이 필요합니다. 문제가 있으면 런처의 디버그 콘솔 로그와 함께 이슈로 제보해주세요.

## 원본과의 차이 (코드)

- `src/main/utils/wine.ts`: Wine/Proton 환경 감지
- PowerShell 매니저: Wine에서 세션 스폰을 시도하지 않고 즉시 실패 반환 (타임아웃 방지)
- 레지스트리 쓰기/삭제: `reg.exe` 폴백 추가
- 프로세스 감시: `Z:\proc` 스캔 폴백 추가
- NSIS 설치파일: 이미 설치된 경우 재설치 대신 런처를 바로 실행하는 스텁 추가 (스팀 바로가기 경로 수정 불필요)
- 자동 업데이트: 이 포크의 Releases(prerelease)에서 받아옵니다
